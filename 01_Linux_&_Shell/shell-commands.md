 # Shell Commands & There Use

 ### 1. Time 
 ```sh
 date # Show current date and time  
 ```

 ### 2. Show item in Directory
```sh
ls # list the file in current directory

ls -a # Show hidden file 

ls -ltr # Show the file details with time stamps
```

### 3. Create & Delete Directory & Files 
```sh
# Directory 
mkdir <directory-name> # Create a folder/directory

rmdir <directory-name> # Remove the directory

rmdir -rf <directory-name> # Remove the directory forcefully

# File
touch # Create a blank file
```
 

### 3. Editor
```sh
vi # Default in all the linux distribution

vim # Advance version of vi editor need to install as per distribution

nano # Advance editor like the vim
```
